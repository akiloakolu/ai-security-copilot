# AI Security Copilot

A SOC-style security analysis tool that helps identify and respond to suspicious activities such as phishing emails, login threats, and malicious scripts.

## 🔍 Features

- Analyze security alerts (phishing, login, PowerShell activity)
- Severity classification (Low / Medium / High)
- Recommended response actions
- SOC-style dashboard UI
- Analysis history tracking
- Real-time frontend ↔ backend communication

## 🛠️ Tech Stack

- Frontend: React (Vite) + Tailwind CSS
- Backend: Node.js + Express
- API Communication: Axios
